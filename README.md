# Happy-places-1 — "Discover and share your favorite places"

## Overview
Happy-places-1 is a class project on Android development that allows users to discover and share their favorite places. This project was created as a learning experience to explore the Android ecosystem and build a functional app. The app enables users to sign up, log in, and create content related to places they have visited.

## Tech Stack
* Primary language: Java
* Android framework
* Firebase Database and Storage for data storage and management
* Material Design components for UI

## Features
* User authentication with sign up and login functionality
* Ability to create content related to places visited, including text and image uploads
* Navigation between activities, including a splash screen, login screen, and content creation screen
* Floating action buttons for easy navigation between activities
* Image selection and upload functionality using Firebase Storage

## Setup & Installation
1. Clone the repository using the following command:
```bash
git clone https://github.com/abdiomari/Happy-places-1.git
```
2. Navigate to the project directory:
```bash
cd Happy-places-1
```
3. Install the necessary dependencies using the following command:
```bash
./gradlew assemble
```
4. Configure the Firebase project by creating a new Firebase project in the Firebase console and adding the Firebase configuration file (`google-services.json`) to the project directory.
5. Run the project on an Android emulator or physical device using the following command:
```bash
./gradlew installDebug
```
6. Start the app on the device by navigating to the app's package name in the device's app drawer.
